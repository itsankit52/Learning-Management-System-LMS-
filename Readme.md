# AI Powered LMS

An **AI Powered Learning Management System (LMS)** built with the **MERN Stack** that supports **Students, Instructors, and Admins**.  
The platform provides **course creation, video lessons, quizzes, progress tracking, payments, real-time chat, certificates, and AI-powered learning tools** like chatbot support, quiz generation, notes summarization, and personalized roadmap generation.

---

## 🚀 Project Details

AI Powered LMS is a full stack online learning platform designed to deliver a modern, interactive, and intelligent learning experience.  
It allows **students to enroll in courses, watch lessons, attempt quizzes, track progress, and use AI-powered tools** for better understanding and faster learning.  
At the same time, **instructors can create and manage courses, upload learning resources, and monitor student engagement**, while **admins can manage the entire platform through analytics and user/course controls**.

The platform is built to combine the core strengths of an LMS with AI-powered productivity and learning support tools.  
It includes features such as:

- **Student, Instructor, and Admin roles**
- **Course creation, management, and enrollment**
- **Video lessons and downloadable study material**
- **Quiz system with score tracking**
- **Progress tracking and continue learning flow**
- **AI chatbot for doubt solving**
- **AI quiz generator**
- **AI notes summarizer**
- **AI roadmap generator**
- **Real-time chat using Socket.io**
- **Course payments using Razorpay**
- **Certificate generation after course completion**
- **Gamification with XP, badges, streaks, and leaderboard**
- **Modern responsive UI with dashboard-based workflow**

The goal of this project is to create a **scalable AI-powered education platform** that goes beyond a basic LMS by adding automation, personalization, and intelligent learning assistance.

## 🛠️ Tech Stack

### Frontend
- **React.js** – for building the user interface
- **Tailwind CSS** – for modern responsive styling
- **React Router DOM** – for frontend routing
- **Axios** – for API communication
- **Context API** – for global state management

### Backend
- **Node.js** – JavaScript runtime for backend
- **Express.js** – backend framework for APIs
- **JWT Authentication** – for secure login and protected routes
- **bcryptjs** – for password hashing
- **Multer** – for handling file uploads
- **Socket.io** – for real-time chat and live features

### Database
- **MongoDB** – NoSQL database for storing users, courses, quizzes, progress, etc.
- **Mongoose** – ODM for MongoDB schema and queries

### Cloud / Storage
- **Cloudinary** – for uploading and managing images, videos, and PDFs

### Payment
- **Razorpay** – for secure course purchase and payment handling

### AI
- **Google Gemini API** – for chatbot, quiz generation, notes summarization, and roadmap generation
